¿Cómo hacer tablas en HTML?

Las tablas son de gran importancia al momento de crear un sitio web debido a que este nos ayuda a dar más estructura.

Si deseas crear tablas vas a necesitar Filas y Columnas. Lo primero que debemos hacer al momento de crear una tabla es ¨invocar¨ el atributo < table >. Para agregar una fila vamos a usar el atributo < tr > y dentro de ello vamos a agregar columnas con el atributo < td >.
